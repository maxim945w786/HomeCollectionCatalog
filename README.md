# HomeCollectionCatalog

## Описание выполненной работы

В рамках практической работы №14 разработан пользовательский интерфейс информационной системы учета домашней коллекции на платформе WPF.

Созданы два основных окна приложения:

* LoginPage.xaml — окно авторизации пользователя.
* RegistrationPage.xaml — окно регистрации нового пользователя.

Для приложения разработан единый фирменный стиль с использованием ресурсов App.xaml. Настроены цвета, шрифты и стили элементов управления.

Реализованы следующие возможности:

* отображение логотипа на форме авторизации;
* отображение логотипа на форме регистрации;
* ввод электронной почты и пароля для авторизации;
* переход между окнами авторизации и регистрации;
* ввод данных пользователя для регистрации;
* проверка совпадения пароля и подтверждения пароля;
* вывод информационных сообщений пользователю;
* использование фирменной цветовой схемы варианта №6;
* использование шрифта Garamond;
* подключение иконки приложения.

## Предметная область

Учет домашней коллекции.

Система предназначена для хранения и каталогизации коллекционных предметов, таких как:

* монеты;
* марки;
* книги;
* виниловые пластинки;
* предметы искусства;
* другие коллекционные объекты.
